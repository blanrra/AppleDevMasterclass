# Showcase — Proyectos de Alumnos

Este espacio es para que los alumnos que completen el curriculum muestren su **Proyecto Integrador** y otras apps creadas durante el aprendizaje.

---

## Como publicar tu proyecto

1. Fork este repositorio
2. Anade tu proyecto a la tabla de abajo
3. Incluye: nombre, descripcion, link al repo, y que modulos aplicaste
4. Abre un Pull Request

## Criterios para aparecer en el Showcase

- El proyecto debe ser original (no una copia del tutorial)
- Debe usar al menos 3 tecnologias del curriculum (SwiftUI, SwiftData, async/await, etc.)
- Debe tener un README en su propio repositorio explicando que hace
- Bonus: publicado en TestFlight o App Store

---

## Inspiracion

Algunos tipos de proyectos que los alumnos han creado:
- Apps de fitness con HealthKit + watchOS
- Gestores de tareas con SwiftData + Widgets
- Apps de viaje con MapKit + CloudKit
- Herramientas de productividad con App Intents + Siri
- Apps con IA on-device usando Foundation Models
