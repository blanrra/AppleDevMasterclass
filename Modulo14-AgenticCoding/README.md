# Modulo 14: Agentic Coding y MCP (Bonus Avanzado)

## Descripcion

El desarrollo ha cambiado. Ya no solo escribes codigo — diriges agentes que lo implementan por ti. Este modulo te ensena las herramientas, metodologias y protocolos que definen el desarrollo profesional en 2025/26: codificacion agentica, Spec Driven Development y Model Context Protocol (MCP).

La joya del modulo: **construiras tu propio MCP Server desde cero**, paso a paso, entendiendo cada linea. No es magia — es un protocolo abierto que puedes dominar.

## Objetivos

Al completar este modulo seras capaz de:

- [ ] Entender que es la codificacion agentica y como integrarla en tu flujo
- [ ] Configurar Claude Code como agente de desarrollo (CLAUDE.md, hooks, skills)
- [ ] Aplicar Spec Driven Development para dirigir agentes
- [ ] Explicar el protocolo MCP: que es, como funciona, por que existe
- [ ] Construir un MCP Server desde cero en Node.js
- [ ] Construir un MCP Server desde cero en Swift
- [ ] Conectar tu MCP Server con Claude Code y otros agentes
- [ ] Entender embeddings y busqueda semantica (NLEmbedding)

## Prerequisitos

- Modulos 00-01 completos (Swift + Arquitectura)
- Tener Claude Code instalado y funcionando
- Conocimiento basico de Node.js (para el MCP en JS)
- Conocimiento basico de terminal y Git

## Mini-Proyecto: Tu Propio MCP Server

Construir un **MCP Server funcional** que:
- Exponga herramientas personalizadas (tools)
- Se conecte con Claude Code
- Lea datos reales (archivos, APIs, bases de datos)
- Sea extensible para tus propios proyectos

> Primero en Node.js (para entender el protocolo) y luego en Swift (para hacerlo nativo Apple).
