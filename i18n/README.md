# Translations / Traducciones

AppleDevMasterclass is written in Spanish by default. Community translations are welcome!

## How to Translate

1. Fork the repository
2. Copy `CLAUDE.md` to `i18n/CLAUDE_[lang].md` (e.g., `CLAUDE_en.md`)
3. Translate the content keeping all technical terms in English
4. Submit a Pull Request

## Available Translations

| Language | File | Status | Contributor |
|----------|------|--------|-------------|
| Espanol (original) | `CLAUDE.md` | Complete | @blanrra |
| English | `i18n/CLAUDE_en.md` | Template | @blanrra |

## Translation Guidelines

- Keep all Swift/Apple technical terms in English (they are universal)
- Translate pedagogical instructions and explanations
- Keep the same structure and section order
- Keep Cupertino MCP commands as-is (they are language-independent)
- Adapt cultural references and analogies to the target language
