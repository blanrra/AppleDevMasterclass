# Modulo 02: Diseno y UX (Semanas 15-16)

## Descripcion

Principios de diseno de Apple: Human Interface Guidelines, el nuevo sistema Liquid Glass, SF Symbols y accesibilidad. Crear apps que se sientan nativas y sean accesibles para todos.
